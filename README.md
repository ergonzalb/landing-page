# Landing Page Project

A modern, conversion-focused landing page with popup lead capture.

## Features

- 🎨 Beautiful gradient background with smooth animations
- 📱 Fully responsive design
- ✨ Scroll reveal animations
- 🎯 Lead capture popup with form
- 🔗 Integration with GlobalControl.io API
- 🎥 Embedded video support
- 🚀 Optimized for conversion

## Setup

1. Replace `YOUR_API_KEY_HERE` in `index.html` with your GlobalControl.io API key
2. Replace placeholder images and video URL with your actual content
3. Customize the copy to match your offer

## Deployment

This project is ready to deploy to Vercel:

```bash
vercel deploy
```

## Customization

- **Colors**: Edit CSS variables in `:root` section
- **Content**: Update text in HTML sections
- **Images**: Replace placeholder URLs with your banner images
- **Video**: Update YouTube embed URL
- **API**: Configure GlobalControl.io API key
